# IoT Smart Home Dashboard

Hệ thống giám sát và điều khiển thiết bị thông minh qua nền tảng Web và giao thức MQTT.

## 🚀 Tính năng chính
- **Dashboard:** Hiển thị biểu đồ nhiệt độ, độ ẩm, ánh sáng thời gian thực.
- **Điều khiển:** Bật/tắt các thiết bị (Đèn, Quạt, Máy bơm) với phản hồi từ phần cứng.
- **Data Sensor:** Xem bảng dữ liệu cảm biến với bộ lọc nâng cao và tìm kiếm linh hoạt.
- **History:** Lưu trữ và tra cứu lịch sử hoạt động của các thiết bị.
- **Profile:** Thông tin cá nhân và tài liệu hướng dẫn.

## 🛠 Công nghệ sử dụng
- **Backend:** Node.js, Express, MySQL.
- **Frontend:** HTML, CSS (Vanilla), JavaScript, TailwindCSS, Chart.js.
- **Giao tiếp:** MQTT (Mosquitto Broker).
